Choosing the right blogging framework

08 November 2015

So, I decided to start my own blog to write posts about the libraries or applications that I publish and there is no better place for a blog to be hosted than our own Github using GithubPages. After creating the repo and posting a Hello World index page, it was time for finding a tool (or framwork) that can facilitate the work of posting to this "blog".

All I wanted was a simple, minimalistic, but yet well styled blog that has a decent content management system so that posting take just a few minutes worth of my time (especially since I have no web UI/UX experience). In my search for such platform I came across jekyll, it sounded promising in many ways. It was simple to config and modify (Ruby and Yamel), has a huge community behind it (>22k starts on Github)
